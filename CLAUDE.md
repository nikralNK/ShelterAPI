# Инструкции для Claude

## Правила работы с кодом
- Не оставлять комментариев в коде
- Не добавлять себя в соавторы в коммитах Git

## Настройки Git
- SSH ключ для подключения к Git имеет постфикс `shelter`

## Описание проекта
- Проект: API для приложения `/var/www/yebitch/TourApp`
- Проект будет развертываться на Windows 10
- Технологический стек:
  - Python 3
  - PostgreSQL
  - Alembic
  - SQLAlchemy
  - FastAPI (with SwaggerUI)

## Авторизация
- Авторизация через JWT токены
- Токены хранятся в PostgreSQL
- Redis не использовать
